# OpenBrain 0.1.0-alpha.1

This initial alpha installs the local engine, generated loader blocks, and Free Mode configuration contract.

## Review after updating

1. Read the generated task in `MIGRATION_TODO.md` if one was created.
2. Run `open-brain doctor` and use `--repair` only after reviewing its report.
3. Run `open-brain scan` to regenerate local indexes.
4. Confirm `interaction.free_mode` is set to the behavior you want. It defaults to `off` when onboarding was skipped.

The update command replaces only the copied engine, the manifest, managed loader blocks, and this migration task file. It does not rewrite vault data or user-authored loader text outside OpenBrain markers.
